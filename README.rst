**Installation**


The PySAD library can be installed via:


.. code-block:: bash

    pip install pysad


Alternatively, you can install the library directly using the source code in this repository by:


.. code-block:: bash

    git clone https://github.com/selimfirat/pysad.git
    cd pysad
    pip install .
